# 第1章 - 节点类型 (Node Types)

本章详细说明肉鸽模式中的各种节点类型、其特性、奖励机制和策略意义。

---

## 1.1 普通战斗节点 (Common Battle)

### 节点特性

| 属性 | 数值 |
|------|------|
| **敌人波数** | 2-3 波 |
| **敌人数量** | 3-6 个僵尸 |
| **难度** | 低 |
| **出现频率** | 最高 |

### 敌人构成

- **敌人等级**: 基础敌人
- **敌人特性**: 无额外词缀
- **推荐植物**: 向日葵 + 豌豆射手组合

### 关卡设置示例

- **第 1-2 关**: 基础僵尸 (普僵、路障僵尸)
  - 血量: 100 HP
  - 攻击力: 20 DPS
  
- **第 3 关**: 引入特殊僵尸 (橄榄球僵尸、舞王僵尸)
  - 橄榄球僵尸: 高血量，低伤害
  - 舞王僵尸: 低血量，可控制其他僵尸

### 奖励机制

**四选二模式**（选择其中2项奖励）:
- **BUFF 3 选 1** - 获得一个随机Buff效果
- **遗物 3 选 1** - 获得一个随机遗物
- **植物选择 3 选 1** - 获得或升级一张植物
- **金币**: 10-30 金

### 战略意义

- **经验积累**: 学习敌人模式的最佳地点
- **资源获取**: 获得基础Buff和遗物
- **植物构筑**: 扩充植物库存

---

## 1.2 精英战斗节点 (Elite Battle)

### 节点特性

| 属性 | 数值 |
|------|------|
| **敌人波数** | 3-5 波 |
| **敌人特性** | 具有1个随机debuff词缀 |
| **血量增幅** | +30% |
| **攻击力增幅** | +20% |
| **难度** | 中 |
| **出现频率** | 中等 |

### 精英敌人示例

#### 示例 1: 护盾 + 路障僵尸
- **特性**: 护盾词缀减伤25%
- **策略**: 需要高穿透或持续伤害
- **推荐植物**: 火焰豌豆、寒冰豌豆

#### 示例 2: 疯狂 + 舞王僵尸
- **特性**: 舞蹈范围扩大50%
- **策略**: 需要快速击杀或控制
- **推荐植物**: 土豆地雷、辣椒

#### 示例 3: 再生 + 普通僵尸
- **特性**: 每5秒恢复100血
- **策略**: 需要持续伤害压制恢复
- **推荐植物**: 豌豆射手、向日葵磨伤

### 奖励机制

**四选三模式**（选择其中3项奖励）:
- **BUFF 3 选 1**
- **遗物 3 选 1**
- **植物选择 3 选 1**
- **金币**: 30-50 金

### 风险与收益

- **难度增加**: 更强的敌人配置
- **奖励更丰富**: 3选2相比普通战2选1获得更多资源
- **构筑验证**: 测试当前构筑对高难敌人的应对能力

---

## 1.3 Boss 节点 (Boss Battle)

### 节点特性

| 属性 | 数值 |
|------|------|
| **敌人波数** | 6 波 |
| **难度** | 极高 |
| **出现频率** | 关键节点（每5关一次） |

### Boss战特点

- **多阶段战斗**: Boss具有多个生命值阶段
- **特殊机制**: Boss具有独特的攻击模式和能力
- **强制遭遇**: 无法避免，必须击败才能继续
- **对构筑的考验**: 检验整个游戏过程中构筑的有效性

### 奖励机制

**四选四模式**（全选所有奖励）:
- **BUFF 3 选 1**
- **遗物 3 选 1**
- **植物选择 3 选 1**
- **金币**: 100+ 金

### 策略意义

- **进度检查点**: 验证构筑的有效性
- **重大奖励**: 获得最丰富的资源
- **构筑转折点**: 基于Boss战结果调整后续路线

---

## 1.4 休息节点 (Rest Node / Campfire)

### 节点特性

| 属性 | 说明 |
|------|------|
| **战斗** | 无战斗 |
| **特点** | 恢复和预定构筑 |
| **出现频率** | 中等 |

### 节点选项

#### 选项 1: 升级一张植物
- **效果**: 选择任意一张已拥有的植物卡升星
- **成本**: 免费
- **用途**: 提升特定植物的强度

#### 选项 2: 恢复阳光池
- **效果**: 增加 100 的初始阳光
- **成本**: 免费
- **用途**: 增加后续战斗的资源可用性

### 战略意义

- **呼吸机会**: 在高强度战斗间隙休整
- **主动构筑**: 根据当前阵容主动升级关键植物
- **资源管理**: 为后续关卡准备充足阳光
- **风险规避**: 避免高难度战斗的一种选择

---

## 1.5 商人节点 (Merchant Shop)

### 节点特性

| 属性 | 说明 |
|------|------|
| **战斗** | 无战斗 |
| **交互** | 购物 |
| **货币** | 局内金币 |

### 店铺物品

#### 1. 植物卡 (20-60 金)
- **选择**: 从卡池中随机 3 选 1
- **用途**: 扩充植物库存
- **成本**: 20-60 金币

#### 2. 植物升级券 (30-80 金)
- **效果**: 无条件升星任意植物 1 次
- **优势**: 无需重复选择该植物卡
- **成本**: 30-80 金币

#### 3. 遗物 (50-150 金)
- **选择**: 随机抽取遗物
- **稀有度**: 稀有度越高越贵
- **成本**: 50-150 金币

#### 4. 铲除服务 (5 金/张)
- **效果**: 移除任意不需要的植物
- **用途**: 清理库存中的弱势植物
- **成本**: 5 金币/张

### 战略意义

- **主动调整**: 允许玩家主动调整卡组
- **资源分配**: 将金币转换为需要的资源
- **构筑纠正**: 修正不理想的前期选择
- **风险决策**: 是否花费稀缺的金币购买遗物

---

## 1.6 事件节点 (Random Event)

### 节点特性

| 属性 | 说明 |
|------|------|
| **出现概率** | 5% |
| **战斗** | 无战斗 |
| **互动** | 做出选择 |

### 事件系统的作用

- **叙事性**: 增加游戏的故事感
- **风险管理**: 提供冒险机制
- **随机性**: 增加游戏的不可预测性
- **转折点**: 可能改变游戏走向的关键节点

### 事件示例详解

#### 事件示例 1: 宝藏陷阱

**描述**: "前方发现一个闪闪发光的箱子...打开它？"

**选项 A: 打开箱子**
- 概率: 50% 获得 2 个遗物，50% 失去 30% 当前阳光
- 期望值: 中等（可能获得大额遗物或失去重要资源）
- 风险: 高

**选项 B: 离开**
- 结果: 获得 50 金
- 期望值: 稳定
- 风险: 低

#### 事件示例 2: 疯狂戴夫的交易

**描述**: "疯狂戴夫提议：交出所有的坚果类植物，我给你一张传说级火焰植物"

**选项 A: 同意**
- 代价: 失去所有坚果墙、高坚果等防守单位
- 获得: 1 个传说遗物
- 适用场景: 已有其他强力防守或专注伤害的构筑

**选项 B: 拒绝**
- 结果: 获得 30 金
- 适用场景: 防守单位是构筑核心

#### 事件示例 3: 神秘雾气

**描述**: "庭院被诡异的雾气笼罩，你的植物似乎在耳语..."

**选项 A: 聆听**
- 结果: 随机升星 2 张植物
- 收益: 可能大幅强化构筑
- 风险: 可能升星不需要的植物

**选项 B: 驱散**
- 结果: 下一关敌人 -20% 血量
- 收益: 稳定的难度降低
- 风险: 一次性收益

### 事件设计原则

- **选择有意义**: 每个选择都有明显的取舍
- **风险回报**: 高风险高回报，保守选择稳定收益
- **戏剧性**: 增加游戏的故事感和参与感

## 节点类型分布建议

| 关卡范围 | 节点分布 |
|----------|----------|
| 1-2关 | 普通战斗 |
| 3-4关 | 普通战斗 + 商店/事件 |
| 5关 | **Boss 战** |
| 6-9关 | 普通战斗 + 精英战斗 + 休息 |
| 10关 | **Boss 战** |
| 11+关 | 精英战斗为主 + 特殊节点 |
