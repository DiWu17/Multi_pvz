## 关键概念速查

### 游戏资源
- **阳光 (☀)**: 用于放置植物的主要资源
- **金币**: 用于商店购买，通过战斗获得
- **遗物**: 全局被动加成，最多获得多个

### 构筑要素
- **植物卡**: 基础战斗单位
- **Buff**: 临时或永久增强效果
- **遗物**: 被动加成系统
- **Debuff**: 敌人强化词缀

### 关键系统
- **星级升级**: 植物升级的主要方式
- **Synergy**: 不同系统之间的组合效应
- **节点系统**: 游戏的路线和奖励机制
- **词缀系统**: 难度调整的主要工具

## 相关参考

### 设计参考来源
- 《杀戮尖塔》- Buff/Debuff组合系统
- 《PVZ 2》- 植物平衡设计
- 《Hades》- 遗物Synergy设计

### 核心设计原则
1. **多样性**: 不同的构筑路线都可行
2. **策略性**: 每次选择都有意义
3. **可重玩性**: 高随机性保证新鲜感
4. **深度**: 复杂的系统交互形成深度
